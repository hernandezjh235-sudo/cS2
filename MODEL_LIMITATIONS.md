# Model limitations — v4.7

- BO3 is a public no-key source and its page/API structure can change.
- Some lower-tier matches may not expose announced five-player lineups or confirmed vetoes before start.
- PandaScore is used only as an optional fixtures/identity fallback unless the configured plan provides more data.
- Detailed official telemetry is strongest when real demos are uploaded or official provider access is obtained.
- A new deployment begins with limited v4.7-specific calibration history.
- Rows remain Pass when no verified player KPR, team identity, match context, or exact market scope is available.
